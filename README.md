# sowrd MOBA Prototype (Unity)

Top-down MOBA prototype built with Unity.
Focus on combat systems, abilities, and scalable architecture.

## Features (WIP)
- Top-down movement & camera
- Basic attack + damage/health system
- One ability per hero (cooldown-based)
- Simple AI (minions/bots)
- Mobile-focused performance (profiling + pooling later)

## Tech
- Engine: Unity
- Language: C#
- Version control: Git + GitHub
